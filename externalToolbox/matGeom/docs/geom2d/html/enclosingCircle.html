enclosingCircle

PURPOSE

ENCLOSINGCIRCLE find the minimum circle enclosing a set of points.

SYNOPSIS

function circle = enclosingCircle(pts)

DESCRIPTION 

ENCLOSINGCIRCLE find the minimum circle enclosing a set of points.

   CIRCLE = enclosingCircle(POINTS);
   compute cirlce CIRCLE=[xc yc r] which enclose all points POINTS given
   as an [Nx2] array.
