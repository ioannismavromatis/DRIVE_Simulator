polygonLength

PURPOSE

POLYGONLENGTH compute perimeter of a polygon

SYNOPSIS

function len = polygonLength(varargin)

DESCRIPTION 

POLYGONLENGTH compute perimeter of a polygon

   L = polygonLength(POLYGON);
   Computes the length of the boundary of a polygon. POLYGON is given by a
   N*2 array of vertices.
