cartesianLine

PURPOSE

CARTESIANLINE create a line with cartesian coefficients

SYNOPSIS

function line = cartesianLine(varargin)

DESCRIPTION 

CARTESIANLINE create a line with cartesian coefficients

   L = cartesianLine(A, B, C);
   Create a line verifying the Cartesian equation:
   A*x + B*x + C = 0;

   See also:
   lines2d, createLine
