curveCMoment

PURPOSE

CURVECMOMENT Compute centered inertia moment of a 2D curve

SYNOPSIS

function m = curveCMoment(curve, p, q)

DESCRIPTION 

CURVECMOMENT  Compute centered inertia moment of a 2D curve
   M = curveCMoment(CURVE, P, Q)

   Example
