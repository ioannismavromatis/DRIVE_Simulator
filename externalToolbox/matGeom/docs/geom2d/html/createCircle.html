createCircle

PURPOSE

CREATECIRCLE create a circle from points

SYNOPSIS

function circle = createCircle(varargin)

DESCRIPTION 

CREATECIRCLE create a circle from points

   C = createCircle(P1, P2, P3);
   Creates a circle going through the 3 given points. 
   C is a 1*3 array of the form : [xc yc radius].

   C = createCircle(P1, P2);
   Creates the circle whith edge [p1 p2] as diameter

   See also:
   circles2d, createDirectedCircle
