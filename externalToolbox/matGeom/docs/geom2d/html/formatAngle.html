formatAngle

PURPOSE

FORMATANGLE Ensure an angle value is comprised between 0 and 2*PI

SYNOPSIS

function alpha = formatAngle(alpha)

DESCRIPTION 

FORMATANGLE  Ensure an angle value is comprised between 0 and 2*PI
   ALPHA2 = formatAngle(ALPHA)
   ALPHA2 is the same as ALPHA modulo 2*PI and is positive.

   Example:
   formatAngle(5*pi)
   ans =
       3.1416

   See also
   vectorAngle, lineAngle
