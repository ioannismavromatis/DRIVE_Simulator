isPointOnPolyline

PURPOSE

ISPOINTONPOLYLINE check if a point belongs to a polyline

SYNOPSIS

function res = isPointOnPolyline(point, poly, varargin)

DESCRIPTION 

ISPOINTONPOLYLINE  check if a point belongs to a polyline

   B = isPointOnPolyline(POINT, POLY)

   Example
       pt1 = [30 20];
       pt2 = [30 10];
       poly = [10 10;50 10;50 50;10 50];
       isPointOnPolyline([pt1;pt2], poly)
       ans =
            0
            1
