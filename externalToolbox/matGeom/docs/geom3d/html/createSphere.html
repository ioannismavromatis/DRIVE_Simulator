createSphere

PURPOSE

CREATESPHERE create a sphere containing 4 points

SYNOPSIS

function sphere = createSphere(varargin)

DESCRIPTION 

CREATESPHERE  create a sphere containing 4 points

   s = createSphere(p1, p2, p3, p4);
   return in s the sphere common to the 4 pointsp1, p2, p3 and p4.
