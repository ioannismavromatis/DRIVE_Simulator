distancePointLine3d

PURPOSE

DISTANCEPOINTLINE3D compute euclidean distance between 3D point and line

SYNOPSIS

function d = distancePointLine3d(point, line)

DESCRIPTION 

DISTANCEPOINTLINE3D compute euclidean distance between 3D point and line

   D = distancePointLine3d(POINT, LINE);
   Returns the distance between point POINT and the line LINE, given as:
   POINT : [x0 y0 z0]
   LINE  : [x0 y0 z0 dx dy dz]
   D     : (positive) scalar  
   
   See also:
   lines3d, points3d
