createRhombododecahedron

PURPOSE

CREATERHOMBODODECAHEDRON create a 3D rhombododecahedron

SYNOPSIS

function [nodes, edges, faces] = createRhombododecahedron()

DESCRIPTION 

CREATERHOMBODODECAHEDRON create a 3D rhombododecahedron

   c = createRhombododecahedron
   c has the form [n, e, f], where n is a 6*3 array with vertices
   coordinate, e is a 12*2 array containing indices of neighbour vertices,
   and f is a 8*3 array containing vertices array of each face.

   See also polyhedra
